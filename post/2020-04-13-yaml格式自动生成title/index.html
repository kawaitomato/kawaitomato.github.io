正文

substr

在default模板里，title也是自动生成的

1

title: "{{ replace .TranslationBaseName "-" " " | title }}"

不过我希望能更自动一点。参照Jekyll的命名方式，我的md文件都采用”yyyy-MM-dd-title”的格式，只要把前11个字符切掉就可以。查了查资料，用substr就可以做到。官方文档是这样描述的

Extracts parts of a string from a specified character’s position and returns the specified number of characters.

Syntax

1

substr STRING START [LENGTH]

并且给出了一个例子

1
2

{{substr "BatMan" 0 -3}} → "Bat"  
{{substr "BatMan" 3 3}} → "Man"

对照着修改，我的就是这样的

1

title: "{{ substr (.TranslationBaseName) 11}}"

title

另外，title的作用是把内容转换成标题大小写，一开始直接复制下来了，后来发现总是会大写首字母才查了查这个功能

Converts all characters in the provided string to title case.

Syntax

1
2
3

title INPUT

{{title "BatMan"}} → "Batman"

Name

.TranslationBaseName的作用是截取文件名的非扩展名和标识符部分

.File.TranslationBaseName

the filename without extension or optional language identifier (e.g., foo)

至于截取其他格式的文件名，参考资料的第三条链接里面有

replace

原模板中replace的作用很容易看出，就是替换，把文件名中的-换成了空格

Replaces all occurrences of the search string with the replacement string.

Syntax

1
2
3
4

replace INPUT OLD NEW

{{ replace "Batman and Robin" "Robin" "Catwoman" }}  
→ "Batman and Catwoman"
